Узнайте, как вести блог в Академическом с помощью блокнотов Jupyter

Установите Python и JupyterLab

Установите Anaconda который включает Python 3 и JupyterLab.

В качестве альтернативы установите JupyterLab с помощью pip3 install jupyterlab.

Создание или загрузка блокнота Jupyter

Выполните следующие команды в терминале, заменив <MY-WEBSITE-FOLDER> и <SHORT-POST-TITLE> на путь к папке вашего академического сайта и краткое название статьи в блоге (используйте дефисы вместо пробелов), соответственно:

mkdir -p <MY-WEBSITE-FOLDER>/content/post/<SHORT-POST-TITLE>/
cd <MY-WEBSITE-FOLDER>/content/post/<SHORT-POST-TITLE>/
jupyter lab index.ipynb

Команда jupyter выше запустит редактор JupyterLab, позволяя нам добавить академические метаданные и написать содержание.

Редактирование метаданных вашего сообщения

Первая ячейка вашего блокнота Jupter будет содержать метаданные вашего сообщения (front matter).

В Jupter выберите Markdown в качестве типа первой ячейки и обведите метаданные Academic тремя тире, указывая на то, что это YAML front matter:

---
title: Название моего поста
date: 2019-09-01

# Поместите сюда любые другие академические метаданные...
---

Редактируйте метаданные своего сообщения, используя документацию в качестве руководства по доступным опциям.

Чтобы установить featured image, поместите изображение с именем featured в папку вашего сообщения.

Другие советы, например, по использованию математики, см. в руководстве writing content with Academic.

Преобразование блокнота в Markdown

jupyter nbconvert index.ipynb --to markdown --NbConvertApp.output_files_dir=.
