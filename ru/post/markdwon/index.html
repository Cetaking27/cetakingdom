Теоретическая информация

Markdown - это легкий язык разметки для создания форматированного текста с помощью обычного текстового редактора. Джон Грубер и Аарон Шварц создали Markdown в 2004 году как язык разметки, привлекательный для человеческого чтения в виде исходного кода. Markdown широко используется в блоггинге, обмене мгновенными сообщениями, онлайн-форумах, программах для совместной работы, страницах документации и файлах readme.

Первоначальное описание Markdown содержало двусмысленности и вызывало вопросы без ответов, в результате чего реализация намеренно и случайно отклонялась от первоначальной версии. Эта проблема была решена в 2014 году, когда давние участники Markdown выпустили CommonMark, однозначную спецификацию и набор тестов для Markdown.

Основная информация о Markdown

Чтобы создать заголовок, используйте знак ( # ), например:

1 # This is heading 1 2 ## This is heading 2 3 ### This is heading 3 4 #### This is heading 4

Чтобы сделать текст жирным, заключите его в двойные звездочки:

    - This text is **bold**.

Чтобы установить курсивный шрифт для текста, заключите его в одинарные звездочки:

    - This text is *italic*.

Чтобы выделить текст жирным шрифтом или курсивом, заключите его в тройные звездочки:

    - This is text is both ***bold and italic***.

Неупорядоченный (маркированный) список может быть отформатирован с помощью звездочки или тире:

1 - List item 1 2 - List item 2 3 - List item 3

Чтобы вложить один список в другой, добавьте отступ для элементов дочернего списка: 1 - List item 1 2 - List item A 3 - List item B 4 - List item 2

Упорядоченный список может быть отформатирован с использованием соответствующих номеров: 1 1. First instruction 2 1. Second instruction 3 1. Third instruction

Чтобы вложить один список в другой, добавьте отступ для элементов дочернего списка: 1 1. First instruction 2 1. Sub-instruction 3 1. Sub-instruction 4 1. Second instruction

Синтаксис Markdown для встроенной ссылки состоит из части [текст ссылки] представляющей текст гиперссылки, и части (file-name.md ) - URL или имя файла на который дается ссылка:

   [link text](file-name.md)

Markdown поддерживает как вставку фрагментов кода в предложение, так и размещение их между предложениями в виде отдельных огороженных блоков. Огражденные блоки кода - это простой способ выделить синтаксис фрагментов кода. Общий формат огороженных блоков кода:

1 language 2 your code goes in here 3

Верхний и нижний индексы:

        $\sin^2 (x) + \cos^2 (x) = 1$

Markwon - это библиотека для Android, которая отображает уценку в виде системных Spannables. Она дает возможность отображать уценку во всех виджетах TextView (TextView, Button, Switch, CheckBox и т.д.), Notifications, Toasts и т.д. WebView не требуется. Библиотека предоставляет разумные настройки по умолчанию для стиля отображения уценки, но также предоставляет все средства для настройки внешнего вида при желании. Поддерживаются все возможности уценки (включая ограниченную поддержку встроенного HTML-кода, таблиц уценки и изображений).
