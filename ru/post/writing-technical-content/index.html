Написание технического контента в формате Markdown

июл. 12, 2019 6 мин. для прочтения
Image credit: John Moeses Bauan

Wowchemy создан для того, чтобы обеспечить беспроблемную работу создателей технического контента. Вы можете сосредоточиться на контенте, а Wowchemy сделает все остальное.

Подсвечивайте фрагменты кода, делайте заметки на уроках математики, рисуйте диаграммы из текстового представления.

На этой странице вы найдете несколько примеров типов технического контента, который может быть отображен с помощью Wowchemy.

Примеры

Код

Wowchemy поддерживает расширение Markdown для подсветки синтаксиса кода. Вы можете настроить стили в опции syntax_highlighter в вашем config/_default/params.yaml file.

```python
import pandas as pd
data = pd.read_csv("data.csv")
data.head()
```

отображается как

import pandas as pd
data = pd.read_csv("data.csv")
data.head()

Mindmaps

Wowchemy поддерживает расширение Markdown для ментальных карт.

Просто вставьте блок кода Markdown markmap и по желанию задайте высоту карты, как показано в примере ниже.

Простая карта ума, определенная как список в формате Markdown:


```markmap {height="200px"}
- Hugo Modules
  - wowchemy
  - wowchemy-plugins-netlify
  - wowchemy-plugins-netlify-cms
  - wowchemy-plugins-reveal
```

отображается как

- Hugo Modules
  - wowchemy
  - wowchemy-plugins-netlify
  - wowchemy-plugins-netlify-cms
  - wowchemy-plugins-reveal

Диаграммы

Wowchemy поддерживает популярный формат Plotly для интерактивных диаграмм.

Сохраните ваш Plotly JSON в папке страницы, например, line-chart.json, а затем добавьте шорткод {{< chart data="line-chart" >}} туда, где вы хотите, чтобы появился график.

Демонстрация:

Вам также может быть полезен Plotly JSON Editor.

Математика

Wowchemy поддерживает расширение Markdown для $\LaTeX$ математики. Вы можете включить эту функцию, переключив опцию math в вашем файле config/_default/params.yaml.

Чтобы отобразить inline или block математику, оберните вашу математику LaTeX с помощью {{< math >}}$...${{< /math >}} или {</* math */>}}$...$${{< /math >}}, соответственно. (Мы обернули математику LaTeX в шорткод Wowchemy math, чтобы Hugo не отобразил нашу математику как Markdown. Шорткод math является новым в версии 5.5-dev.)

Пример математического блока:

{{< math >}}
$$
\gamma_{n} = \frac{ \left | \left (\mathbf x_{n} - \mathbf x_{n-1} \right )^T \left [\nabla F (\mathbf x_{n}) - \nabla F (\mathbf x_{n-1}) \right ] \right |}{\left \|\nabla F(\mathbf{x}_{n}) - \nabla F(\mathbf{x}_{n-1}) \right \|^2}
$$
{{< /math >}}

отображается как

$$\gamma_{n} = \frac{ \left | \left (\mathbf x_{n} - \mathbf x_{n-1} \right )^T \left [\nabla F (\mathbf x_{n}) - \nabla F (\mathbf x_{n-1}) \right ] \right |}{\left \|\nabla F(\mathbf{x}_{n}) - \nabla F(\mathbf{x}_{n-1}) \right \|^2}$$

Example inline math {{< math >}}$\nabla F(\mathbf{x}_{n})${{< /math >}} renders as $\nabla F(\mathbf{x}_{n})$.

Пример многострочная математика с использованием математического перевода строки (\\\\):

{{< math >}}
$$f(k;p_{0}^{*}) = \begin{cases}p_{0}^{*} & \text{if }k=1, \\
1-p_{0}^{*} & \text{if }k=0.\end{cases}$$
{{< /math >}}

отображается как

$$ f(k;p_{0}^{*}) = \begin{cases}p_{0}^{*} & \text{if }k=1, \\ 1-p_{0}^{*} & \text{if }k=0.\end{cases} $$

Диаграммы

Wowchemy поддерживает расширение Markdown для диаграмм. Вы можете включить эту функцию, переключив опцию diagram в вашем файле config/_default/params.toml или добавив diagram: true в основной текст вашей страницы.

An example flowchart:

```mermaid
graph TD
A[Hard] -->|Text| B(Round)
B --> C{Decision}
C -->|One| D[Result 1]
C -->|Two| E[Result 2]
```

отображается как

graph TD A[Hard] -->|Text| B(Round) B --> C{Decision} C -->|One| D[Result 1] C -->|Two| E[Result 2]

Пример диаграммы последовательности:

```mermaid
sequenceDiagram
Alice->>John: Hello John, how are you?
loop Healthcheck
    John->>John: Fight against hypochondria
end
Note right of John: Rational thoughts!
John-->>Alice: Great!
John->>Bob: How about you?
Bob-->>John: Jolly good!
```

отображается как

sequenceDiagram Alice->>John: Hello John, how are you? loop Healthcheck John->>John: Fight against hypochondria end Note right of John: Rational thoughts! John-->>Alice: Great! John->>Bob: How about you? Bob-->>John: Jolly good!

Списки Todo

Вы можете писать списки дел в формате Markdown:

- [x] Write math example
  - [x] Write diagram example
- [ ] Do something else

отображается как

  • Write math example
    • Write diagram example
  • Do something else

Таблицы

Сохраните вашу таблицу в виде CSV-файла в папке вашей страницы, а затем отобразите ее, добавив шорткод Table на вашу страницу:

{{< table path="results.csv" header="true" caption="Table 1: My results" >}}

отображается как

customer_id score
1 0
2 0.5
3 1
Table 1: My results

Callouts

Academic поддерживает shortcode for callouts, также называемые asides, hints, или alerts. Обернув абзац в `{{% callout note %}} … {{% /callout %}}}, он будет отображаться как примечание.

``markdown {{% /callout note %}} Markdown aside полезен для отображения уведомлений, подсказок или определений для ваших читателей. {{% /callout %}}


отображается как



<div class="alert alert-note">
  <div>
    Отступ в Markdown полезен для отображения уведомлений, подсказок или определений для ваших читателей.
  </div>
</div>


### Спойлеры

Добавьте спойлер на страницу, чтобы раскрыть текст, например, ответ на вопрос, после нажатия кнопки.

``markdown
{{< spoiler text="Нажмите для просмотра спойлера" >}}
Вы нашли меня!
{{< /spoiler >}}

отображается как

Нажмите для просмотра спойлера

Ты нашел меня!
